✨ 核心特性

🧠 本体论驱动

  • YAML 定义实体关系
  • 一次调用多跳查询
  • 自动路径规划
  • 并行数据获取

🚦 三层路由设计

  • QueryAgent - 查询意图
  • InvestigateAgent - 排查意图
  • Admin - 引导式对话
  • LLM 智能意图识别

⚡ DirectOutput 机制

  • 绕过 LLM 二次处理
  • 性能提升 4-10 倍
  • 数据查询专属优化
  • 降低 Token 消耗

🔌 MCP 集成

  • Spring AI MCP 协议
  • 可扩展工具集
  • 标准化工具接口
  • 跨平台兼容
