# 任务队列 - 通宵执行

## 执行规则（必须遵守）
1. 找到第一个 `[ ]` 开头的任务
2. 执行该任务
3. 完成后执行：`git add -A && git commit -m "完成: 任务名"`
4. 把任务的 `[ ]` 改成 `[x]`
5. 输出 `<promise>NEXT</promise>`
6. 如果所有任务都是 `[x]`，输出 `<promise>ALL_DONE</promise>`

## 重要提示
- 每个任务独立完成，不依赖其他任务的上下文
- 如果遇到阻塞问题，在任务后面添加注释说明，然后继续下一个
- 优先保证代码能编译通过
